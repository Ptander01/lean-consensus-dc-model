# Scripts

Python scripts for the AI Data Center Consensus GIS project.

## Data Ingestion Scripts

- `ingest_dch.py` - Import DataCenterHawk data
- `Ingest_Synergy.py` - Import Synergy data
- `Ingest_SemiAnalysis.py` - Import time-series forecasts (2023-2032)
- `Ingest_NPM.py` - Import NewProjectMedia data
- `Ingest_Woodmac.py` - Import WoodMac data

## Processing Scripts

- `Campus_Rollup_new.py` - Aggregate buildings to campus level
- `ETL Script.py` - Main ETL workflow

## Quality Assurance

- `QA_Region_Country.py` - Validate region and country fields
- `QA_Validation_Final.py` - Final data quality checks

## Utilities

- `helper_Scripts.py` - Common utility functions

## Schema Exports

See `Schema_Exports/` subfolder for complete schema documentation in multiple formats.
